## Summary

Inquiry Confirmation And Status helps users know when their owner inquiry has been sent and whether a listing is still accepting contact. It matters because users need confidence that their action was completed and a clear signal when they must try another option. This feature closes the loop between contact intent and confirmed next step.

## User Story

As a User, I want to see confirmation and status for my inquiry so that I know whether my message was sent and whether I can still follow up on that listing.

## Acceptance Criteria

- After submitting an inquiry, the user sees a clear confirmation that the message was sent.
- The confirmation identifies the listing the inquiry was sent for.
- The user can see whether the listing is currently open or closed to inquiries.
- If the listing is no longer accepting inquiries, the user sees a clear status message and cannot continue with that contact action.
- The user can return to browsing other listings from the confirmation or status view.
- The user can tell whether the inquiry is in a successful state without needing technical details.

## Priority

High - Inquiry confirmation and status are essential for trust, user confidence, and next-step decision making after contact.

## Dependencies
