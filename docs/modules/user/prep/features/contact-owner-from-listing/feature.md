## Summary

Contact Owner From Listing allows authenticated users to initiate direct inquiry from a selected PG listing when they are ready to take action. It matters because discovery only creates value when users can convert interest into real owner contact without leaving the flow. This feature shortens the path from evaluation to inquiry and supports faster decision-making.

## User Story

As a User, I want to contact the owner directly from a listing so that I can quickly express interest and move forward with the right PG option.

## Acceptance Criteria

- From a listing detail or result card, the user can start a contact action for that specific listing.
- The contact flow clearly identifies which listing the inquiry is being sent for.
- The user can submit an inquiry and receive a visible confirmation that it was sent.
- The user can return to browsing after contact submission without losing core search context.
- If the listing is not available for inquiry, the user sees a clear message and cannot proceed with contact.
- Direct contact from listing is available only to authenticated users.

## Priority

High - Direct owner contact is the primary conversion step from discovery to meaningful user outcome.

## Dependencies

none
