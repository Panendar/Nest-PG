# Success Metrics — PG Management Platform

## Weekly Metrics
- Website/App visitors
- Search usage (radius vs city)
- Registered users
- Registered PG owners
- PG listings created
- PG listings viewed
- Contact requests sent
- Successful owner-user interactions
- Active listings

## Derived Metrics
- Visitor → Registration rate (%)
- User → Contact conversion rate (%)
- Owner → Listing creation rate (%)
- Listing → Contact rate (%)
- Average search radius used
- City-based vs radius-based usage (% split)

## Engagement Metrics
- Average session time
- Listings viewed per session
- Repeat visits per user
- Owner activity frequency

## What Success Looks Like

### User Experience
- “I can easily find PGs near me or in any city”
- “I can compare multiple PGs quickly”
- “Contacting owners is simple”

### Owner Experience
- “I get consistent inquiries”
- “Listing and updating PG is easy”
- “Platform brings real leads”

### Growth
- Increasing listings week over week
- Increasing contact requests
- Balanced growth in users and owners

### Platform Health
- Low listing churn
- High active listing ratio
- Stable response time (< 3s)
