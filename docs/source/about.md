# PG Management Platform — Overview

## Core Idea
A centralized platform where students can discover, compare, and contact PG accommodations without physically visiting multiple locations.

## What is it?
A location-aware PG discovery and listing system with:
- User and Owner authentication
- Real-time PG listing and updates
- Radius-based and city-based search
- Media-rich listings (photos, details)
- Direct contact functionality

## Core Features

### For Users
- Search PGs by:
  - Radius (default: 3 km, adjustable)
  - City/Town selection
- View:
  - Photos
  - Pricing
  - Amenities
  - Location map
- Contact PG owners directly

### For Owners
- Register/Login
- Add/Edit PG listings
- Upload photos
- Update availability in real-time

### System Capabilities
- Geolocation filtering
- Real-time data updates
- Authentication (Users & Owners)
- Scalable listing system

## Value Proposition

### Users
- Saves time and effort
- Centralized discovery
- Better comparison of options

### Owners
- Increased visibility
- Direct customer reach
- Easy listing management

## USP
- Radius + City hybrid search
- Real-time PG updates
- Dual-role platform (Users + Owners)
- Focused student-friendly interface
